# Weather Dashboard 🌦️

A modern Weather Dashboard built using Flask, OpenWeatherMap API, Bootstrap 5, HTML, CSS, and JavaScript.

## Features

* Search weather by city
* Real-time temperature
* Humidity information
* Wind speed details
* Feels like temperature
* Sunrise and sunset timings
* Weather icons
* Recent search history
* Dynamic weather themes
* Weather tips
* Air quality section UI
* 5-day forecast UI
* Responsive design
* Loading spinner

## Technologies Used

* Python
* Flask
* OpenWeatherMap API
* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/weather-dashboard.git
```

Move into project folder:

```bash
cd weather-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
python app.py
```

Open browser:

```text
http://127.0.0.1:5000
```
